Russell's URL Shortener

User stories

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

https://lit-headland-86744.herokuapp.com/new/https://www.google.com
https://lit-headland-86744.herokuapp.com/new/https://www.github.com/russelltchang

Example creation output:

{"original_url" : "https://www.github.com/russelltchang",
"short_url" : "https://lit-headland-86744.herokuapp.com/1523145782803"}

Usage:

https://lit-headland-86744.herokuapp.com/1523157922786

Will redirect to:

https://www.github.com/russelltchang
